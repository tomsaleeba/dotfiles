# Useful keybindings
Manjaro uses `dunst` for notifications, and if you miss one, you can recall the
last 20 items. The binding is configured in `~/.config/dunst/dunstrc`:
```
    history = ctrl+mod4+h
```
Note that `mod4` is the `gui/windows` key.
